# Tkinter To-Do List Application

A simple desktop To-Do List application built with Python and Tkinter.

This project allows users to add tasks, delete tasks by entering their number, and automatically saves all tasks to a file so they persist between application runs.

---

## Features

- Add new tasks using a text input and button
- Display tasks with automatic numbering
- Delete tasks by entering the task number
- Tasks are saved to a local file
- Tasks are loaded automatically when the application starts
- Simple and clean graphical user interface

---

## Technologies Used

- Python
- Tkinter
- File I/O (Text File Handling)

---

## How to Use

1. Enter a task in the input field
2. Click the **Add** button to add the task to the list
3. Tasks are displayed with numbers
4. To delete a task, enter its number and click **Delete**
5. Tasks are saved automatically and restored when the app is reopened
